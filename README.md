# Дипломный проект по автоматизированому тестированию
Тестирование веб-приложения для заказа бургеров Stellar Burgers
## Задание 1: Юнит-тесты
### Что нужно сделать 
- Покрыть тестами класс Burger
- Использовать моки и параметризацию
- Проверить покрытие с помощью JaCoCo
### В проекте используется
Java 11, Maven, JUnit 4, Mockito, JaCoCo
## Запуск тестов
mvn clean test
## Получение отчёта
1. mvn verify
2. в папке target/site/jacoco/ найти файл index.html, нажать на него правой кнопкой мыши >> Open In >> Browser >> браузер для просмотра
