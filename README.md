# Stellar Burgers Unit Testing

## Описание проекта

Stellar Burgers - веб-приложение для сборки и оформления заказов бургеров с регистрацией и отслеживанием статуса заказов.

### Цель проекта

Покрыть юнит-тестами ключевой класс Burger

### Тип тестирования

Unit testing

### Реализованные задачи

- Разработаны юнит-тесты класса Burger.
- Использован Mockito для мокирования зависимостей.
- Применена параметризация для тестов.
- Проведена проверка покрытия кода с помощью JaCoCo.

## Структура проекта

```text
project/
    ├─ main/
    │   └─ java/     #классы
    └─ test/
        └─ java/     #тесты
```

## Технологии

- Java 11
- Maven
- JUnit 4
- Mockito
- JaCoCo

## Конфигурация и запуск

### Запуск тестов

*mvn clean test*

### Получение отчёта

*mvn verify*
